Installation Mac OS X:
**********************

.. code-block:: bash

    brew install libxml2 libxslt

    sudo pip install virtualenv

Usage:
******

.. code-block:: bash

    virtualenv env
    source env/bin/activate
    pip install -r requirements.txt
    python main.py
